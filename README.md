# hedaAssistantGit
一个完成一半的 demo  河大助手

课表考试安排,简单的新闻,推送功能,好长时间以前的代码了挖了出来.给新手们学习一下
 服务端用 php 写的数据库是 mysql  基于 phlapi 框架
 
 ios 客户端 Object-C  主框架基于    [ CCNavtab ](https://github.com/xiongcaichang/CCNavTab "Title") 一个数组快速搭建UI框架
